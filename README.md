# Dropout TV Kodi Plugin

This is a work in progress attempt to create a
[KODI TV](https://kodi.tv/)
plugin allowing to watch
[Dropout TV](https://watch.dropout.tv/help).

Requires a valid subscription to droput.tv

# Developing

## Tooling

```sh
# Validate addon.xml and settings.xml
docker-compose -f dev/tooling/docker-compose.yml up --build
```
